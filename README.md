## Getting Started
### Prerequisites
```bash
node.js (v14 or higher)
npm or yarn
```

### Installation
1. Clone the repository
```bash
git clone https://github.com/coderTtxi12/grozan.git
cd grozan
```

2. Install dependencies
```bash
npm install
```

3. Run the application
```bash
npm start
```
